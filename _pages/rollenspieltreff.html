Das Paladins Inn ist ein Rollenspieltreffpunkt in Bensheim. Durch die Covid-19-Pandemie leider gebremst, wird es voraussichtlich im Mai 2023 die Türen öffnen.

Auf 50 qm bietet es Platz für Rollenspielgruppen. Außerdem haben die Gäste Zugriff auf die Rollenspielbibliothek des Hauses.

Natürlich hat das Paladins Inn eine virtuelle Dependence auf Discord. Hier kann man sich auch schon vorher zum Rollenspielen oder einfach nur zum Erfahrungsaustausch treffen.

Außerdem lässt sich der Paladins Inn Server auf Discord auch für virtuelle Conventions "leihen". Falls es einer Con-Orga nicht möglich ist, einen eignenen Server aufzusetzen oder sie die Kosten für das "boosten" nicht tragen will, findet hier einen vollständig konfigurierten Server vor auf dem für die Dauer des Cons auch Adminrechte für die Orga vergeben werden.

Soweit es uns möglich ist, unterstützen wir auch Orgas technisch bei der Einrichtung eigener Server oder bieten andere Infrastruktur wie den Twitch-Channel oder Hilfe beim Editing oder Streaming. Fragt uns einfach um Hilfe!
